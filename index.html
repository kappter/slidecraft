Welcome to SlideCraft

SlideCraft is designed for students and educators to create interactive process guides. Follow these steps:

  1. Select a theme (Light or Dark) for best visibility.
  2. Upload a CSV file with steps (Step, Description, Order Number, Image URL, Duration).
  3. Check 'Auto-Advance' to move slides automatically based on duration (default 5 seconds).
  4. Click 'Start' to begin the guided process.
  5. Complete the quiz and generate a PDF report.

Our vision: Empower learning with simple, mobile-friendly tools for process mastery.
